## طريقة الرفع على Vercel (5 دقائق)

### الخطوة 1 — رفع على GitHub
1. افتح github.com وسجل دخول
2. اضغط **New repository**
3. اسم المشروع: `syrian-fund-training`
4. اضغط **Create repository**
5. ارفع كل الملفات (drag & drop أو upload files)

### الخطوة 2 — ربط مع Vercel
1. افتح vercel.com وسجل دخول بحساب GitHub
2. اضغط **New Project**
3. اختر المشروع `syrian-fund-training`
4. اضغط **Deploy**
5. انتظر دقيقة واحدة ← رابطك جاهز! 🎉

## تشغيل محلياً (اختياري)
```bash
npm install
npm run dev
```
